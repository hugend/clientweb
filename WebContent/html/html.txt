4. 양식태그

	1)클라이언트가 서버와 통신하기 위해 필요한 태그 이 양식 태그를 통해 선택하거나 입력하는 값을 서버로 전송하기 위해서는
	무조건 <form></form>태그 사이에 선언해야한다.
	양시개그에서 선택하거나 입력한 값이 <form>과 </form>d안에 정의되어 있어야만 서버로 전송된다.
	그렇지 않은 경우 서버로 전송되지 않는다.
	<form>태그의 속성에느s action, method, enctype..등이 있으며
	method 는 요청방식을 명시하고
	action에는 요청할 web application의 path를 정의한다.
	2)양식태그를 정의하는 경우 name 속성을 정의해야 서버로 전송되는 값을 구분할 수 있다.
